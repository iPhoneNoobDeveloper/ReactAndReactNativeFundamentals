# Day 01: Modern JavaScript Fundamentals (ES6+)

This day focuses on building a strong foundation in **modern JavaScript**
required for **React** and **React Native** development.

If you come from an iOS background, think of this as learning the
“Swift language layer” before SwiftUI or UIKit.

---

## Concepts Covered

### Core JavaScript
- `let` vs `const`
- Dynamic typing and common JS quirks
- Strict equality (`===`) vs loose equality (`==`)

### Functions
- Function declarations vs expressions
- Arrow functions
- Closures and callbacks

### Strings & Objects
- Template literals
- Objects and nested objects
- Object and array destructuring

### Immutability (Very Important for React)
- Spread operator for objects and arrays
- Shallow vs nested copies

### Arrays (Used everywhere in React)
- `map`
- `filter`
- `reduce`
- Method chaining
