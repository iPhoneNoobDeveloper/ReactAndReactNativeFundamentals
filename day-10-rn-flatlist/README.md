# Day 10: React Native Lists with FlatList

This day focuses on efficient list rendering in React Native.

## Concepts Covered
- `FlatList`
- `data`
- `renderItem`
- `keyExtractor`
- Reusable row components
- `ListHeaderComponent`
- `ItemSeparatorComponent`
- Basic list performance

## Key Learning

Use `FlatList` instead of `.map()` for large React Native lists.

`FlatList` is conceptually similar to:
- `UITableView`
- `UICollectionView`
