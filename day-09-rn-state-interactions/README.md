# Day 09: React Native State & Interactions

This day focuses on interactive React Native UI using state.

## Concepts Covered
- `useState` in React Native
- Controlled `TextInput`
- `Switch` for boolean state
- `Pressable` for custom buttons
- `Alert`
- Live preview UI
- Immutable object state updates

## Key Learning

React Native uses the same React state model as React Web.

Only the UI primitives change:
- Web input becomes `TextInput`
- Web button becomes `Pressable` / `Button`
- Web checkbox becomes `Switch`
